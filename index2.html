Cara Kerja Web DATA TERNAK

  • User login menggunakan token (xpromilk) yang diberikan admin.
  • Tambah ternak baru dengan detail lengkap (foto, nama, tanggal masuk, harga beli).
  • Catat semua biaya pakan, obat, kandang, per ternak.
  • Tambahkan bobot ternak dengan tanggal & kg.
  • Jual ternak, sistem otomatis hitung profit/rugi.
  • Buat laporan bulanan PDF/CSV, termasuk grafik pengeluaran vs profit.
  • Token salah muncul notif error dan bisa hubungi admin via WhatsApp.
